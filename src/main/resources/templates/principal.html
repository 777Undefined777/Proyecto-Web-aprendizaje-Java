Estructuras de Datos

Las estructuras de datos son fundamentales para la programación. Aquí hay algunos ejemplos de estructuras de datos en Java.

Ejemplo de una lista enlazada:


// Implementación de una lista enlazada simple en Java
class Nodo {
    int data;
    Nodo next;

    Nodo(int d) {
        data = d;
        next = null;
    }
}

public class ListaEnlazada {
    Nodo head;

    public void insertar(int data) {
        Nodo nuevoNodo = new Nodo(data);
        nuevoNodo.next = head;
        head = nuevoNodo;
    }

    public void imprimirLista() {
        Nodo n = head;
        while (n != null) {
            System.out.print(n.data + " ");
            n = n.next;
        }
    }

    public static void main(String[] args) {
        ListaEnlazada lista = new ListaEnlazada();
        lista.insertar(1);
        lista.insertar(2);
        lista.insertar(3);
        lista.imprimirLista(); // Salida: 3 2 1
    }
}

Este programa crea una lista enlazada simple y la imprime.

Programación Orientada a Objetos (POO)

La programación orientada a objetos (POO) es un paradigma de programación basado en el concepto de "objetos". Aquí hay algunos ejemplos de POO en Java.

Ejemplo de una clase y objeto en Java:


// Clase Persona con atributos y métodos
class Persona {
    String nombre;
    int edad;

    // Constructor
    Persona(String nombre, int edad) {
        this.nombre = nombre;
        this.edad = edad;
    }

    // Método para mostrar información
    void mostrarInfo() {
        System.out.println("Nombre: " + nombre);
        System.out.println("Edad: " + edad);
    }

    public static void main(String[] args) {
        Persona persona1 = new Persona("Jorge", 25);
        persona1.mostrarInfo(); // Salida: Nombre: Jorge Edad: 25
    }
}

Este programa define una clase Persona y crea un objeto de esa clase.
